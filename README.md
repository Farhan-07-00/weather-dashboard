# 🌤️ WeatherIQ – AI-Powered Weather Intelligence Dashboard

WeatherIQ is a modern, responsive weather intelligence platform that provides real-time weather information, air quality insights, interactive maps, forecasts, and personalized recommendations. Built using JavaScript, OpenWeather APIs, Leaflet Maps, and Chart.js, the application delivers an engaging user experience with dynamic weather-based video backgrounds and smart weather analytics.

## 📸 Features

### 🌦 Real-Time Weather
- Current temperature and weather conditions
- Feels-like temperature
- Humidity, wind speed, pressure, and visibility
- Local time based on selected location

### 📍 Location Services
- Search weather by city name
- Use current device location
- Interactive weather map with location marker

### 🗺 Interactive Weather Maps
- Temperature Layer
- Cloud Layer
- Precipitation Layer
- Wind Layer
- Pressure Layer

### 📊 Forecast & Analytics
- 24-Hour Temperature Forecast
- 5-Day Weather Forecast
- Interactive temperature charts using Chart.js

### 🌫 Air Quality Monitoring
- AQI Index
- PM2.5
- PM10
- NO₂
- SO₂
- O₃
- CO

### 🧠 AI Weather Insights
- Automated weather analysis
- Personalized recommendations
- Smart weather alerts

### 👕 Lifestyle Recommendations
- Clothing suggestions based on weather
- Outdoor activity suitability scores
- Running score
- Cycling score
- Photography score
- Outdoor comfort score

### 🌅 Sun Journey Tracker
- Sunrise and sunset timings
- Dynamic daylight progress visualization
- Countdown to sunrise/sunset

### 🎤 Voice Search
- Search cities using voice commands

### ❤️ Favorite Cities
- Save frequently viewed cities
- Quick access to favorite locations

### 🎥 Dynamic Video Backgrounds
- Sunny weather videos
- Rain animations
- Snow backgrounds
- Cloudy weather scenes
- Automatic switching based on current conditions

### 🎨 Modern UI
- Day/Night theme toggle
- Responsive design
- Glassmorphism-inspired interface
- Mobile-friendly layout
