## 📖 Overview

WeatherIQ is a modern weather intelligence platform designed to provide real-time weather forecasting, air quality monitoring, interactive weather maps, AI-generated weather insights, and personalized recommendations.

The application combines weather visualization, environmental analytics, and intelligent recommendations into a single user-friendly dashboard.

---

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Farhan-07-00/weather-dashboard.git
```

### Navigate to Project

```bash
cd weather-dashboard
```

### Configure API Key

Create:

```javascript
const OPENWEATHER_API_KEY = "YOUR_API_KEY";
```

inside:

```text
config.js
```

### Run Project

Open:

```text
index.html
```

or use VS Code Live Server.
